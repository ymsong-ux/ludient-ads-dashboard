 Lululab 광고 운영 대시보드 - Windows 실행 안내
═════════════════════════════════════════════

【사전 준비】 Python 설치 (한 번만)

1. https://www.python.org/downloads/ 접속
2. "Download Python 3.x" 노란 버튼 클릭 후 다운로드
3. 설치 파일 실행
4. ★★ 중요 ★★
   "Add Python to PATH" 체크박스 반드시 체크
   (안 하면 명령어 인식 안 됨)
5. "Install Now" 클릭

이미 Python이 설치되어 있으면 이 단계 생략.


【실행 방법】

1. 이 폴더의 "install_and_run.bat" 더블클릭
2. 검은색 명령 창이 뜨면서 자동 설치 시작
3. 1~2분 후 브라우저에 대시보드 자동 표시

종료: 검은색 명령 창 닫기 (X 버튼)


【문제 발생 시】

▶ "Python이 설치되어 있지 않습니다" 에러
  → 위 사전 준비 단계의 PATH 체크 다시 확인 후 재설치

▶ 보안 경고 ("Windows에서 PC를 보호했습니다")
  → "추가 정보" 클릭 → "실행" 클릭

▶ 패키지 설치 실패
  → 인터넷 연결 확인 후 재시도

▶ 그래도 안 되면
  → 송영민에게 문의 (ym.song@lulu-lab.com)
